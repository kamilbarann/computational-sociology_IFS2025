# Computational Sociology — Course Labs

Agent-based social simulation models implemented in Python.

> Course: **Computational Sociology** · Social Informatics, AGH UST (2025)

## Models

| # | Folder | Model | Description |
|---|---|---|---|
| 01 | `01_voter-model` | Voter Model | Binary opinion dynamics on a network |
| 02 | `02_latane-model` | Latané Model | Social impact theory simulation |
| 03 | `03_knowledge-transfer-model` | Knowledge Transfer | Information diffusion in social networks |
| 04 | `04_heider-model` | Heider Balance | Structural balance theory in signed graphs |

## Tech Stack

- Python 3
- NumPy · Matplotlib · NetworkX
